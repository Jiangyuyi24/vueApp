vue.js仿饿了么外卖App
=====================
主要技术：<br />
-----------------------
* 使用Webpack前端构建工具；<br />
* 使用Vue-cli脚手架初始化Vue.js项目；<br />
* es6语法编写js代码；<br />
* 使用stylus预处理框架编写css；<br />
* 使用vue-router实现路由功能；<br />
* 使用vue-resource与后端数据交互；<br />
* 使用Vue.js过渡效果编写交互动画；<br />
* 制作并使用图标字体，减少请求；<br />
* 使用css sticky footer经典布局；<br />
* 使用flex弹性布局，适应移动端；<br />
